# JSHawk Privacy Policy

Last updated: April 2026

## Summary
JSHawk collects no user data. All processing is local to your browser.

## Data Collection
JSHawk does not collect, store, transmit, or share any personal data, browsing history, credentials, or any other user information with any external party or server.

## Local Storage Only
- **chrome.storage.session** — scan findings are stored locally per-tab and automatically deleted when the browser closes
- **chrome.storage.local** — user preferences (entropy threshold, custom patterns) stored on your device only

## What JSHawk Reads
JSHawk reads JavaScript file content and API endpoint responses from sites you choose to scan. This content is analyzed locally for security vulnerabilities and is never transmitted outside your browser.

## No External Servers
JSHawk has no backend server. No analytics, no tracking, no telemetry, no advertising.

## Authorized Use Only
JSHawk is intended for authorized security testing only. Users are responsible for ensuring they have permission to test any system they scan.
